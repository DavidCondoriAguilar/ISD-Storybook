# 🏭 Gestión de Producción - Dashboard

Este es el panel administrativo para gestionar y visualizar la data exportada desde la aplicación móvil de producción.

## 🚀 Funcionalidades

- **Importación JSON**: Carga los archivos `.json` compartidos desde la app móvil.
- **Estadísticas en Tiempo Real**: Visualización de producción total, tiempos y eficiencia.
- **Historial Completo**: Registro detallado de cada entrada de producción.
- **Gestión por Trabajador**: Visualiza el rendimiento individual de cada operario.
- **Persistencia Local**: No requiere login, los datos se almacenan en tu navegador.

## 📂 Cómo usar

1. Exporta los registros desde la **App Móvil de Producción**.
2. Comparte el archivo JSON a tu computadora (WhatsApp, Correo, etc.).
3. En este Dashboard, haz clic en **"Importar Registro"**.
4. Selecciona el archivo JSON y ¡listo! Los datos se cargarán automáticamente.

## 🛠️ Tecnologías

- **React + Vite**
- **Recharts**: Para visualización de datos.
- **Lucide React**: Iconografía moderna.
- **Framer Motion**: Animaciones premium.
- **Vanilla CSS**: Diseño moderno y responsivo.
